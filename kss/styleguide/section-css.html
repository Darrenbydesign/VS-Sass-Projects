2.1 #CSS - Typography Typography

Headings, paragraphs, blockquotes, lists, and more have some global resets. The resets are handled by the vendor styles from Boostrap. We modify the sizing and font-family here to accomodate the project needs

2.1.1 #CSS - Typography - Body Text Body Text

This is where we modify the base text size for the entire site. This is modifiable on a theme basis.
