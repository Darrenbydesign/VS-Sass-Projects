1.1 #Components - Buttons Buttons

Use the button classes on an a, button, or input element.

Example 
<div class="form-group">
  <button class="button" type="submit">Button</button>
  <input class="button" type="button" value="Input">
  <input class="button" type="submit" value="Submit">
</div>
<div class="form-group">
  <button class="button-small">button-small</button>
</div>
<div class="row">
  <div class="form-group col-md-6">
    <button class="button-block">Button block</button>
  </div>
</div>
<div class="form-group">
  <button class="button-text" type="submit">Button</button>
  <input class="button-text" type="button" value="Input">
  <input class="button-text" type="submit" value="Submit">
</div>
